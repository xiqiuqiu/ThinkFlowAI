# ThinkFlow-UI-Prompt

This document contains the UI design prompt used to create the OmniMind interface, inspired by terminal aesthetics and modern minimalist design.

---

## English Version

**Prompt:**

Create a modern, geeky, and minimalist web interface inspired by terminal aesthetics and node-based workflows.

- **Background:** Use a clean white or very light gray background with a subtle dot or line grid pattern (reminiscent of a blueprint or a code editor canvas).
- **Typography:**
  - **UI & Labels:** High-quality Monospace font (e.g., JetBrains Mono, Fira Code) for that "developer" feel.
  - **Headings:** Bold Sans-Serif font (e.g., Inter, Geist) for high readability and professional contrast.
- **Color Palette:**
  - **Main BG:** Pure White (#FFFFFF).
  - **Text:** Deep Slate Gray (#1E293B) for primary content.
  - **Accents:** Warm Orange/Peach (#F97316 or #FB923C) for interactive states, highlights, and primary buttons.
  - **Subtle Elements:** Light Slate (#F1F5F9) for backgrounds of secondary components.
- **Key Components:**
  - **Window-Style Nodes:** Interactive cards or nodes designed like simplified OS windows, featuring a title bar with the classic "traffic light" dots (Red, Yellow, Green) on the top-left.
  - **Terminal Input:** Search bars and input fields should use command-line metaphors, such as starting with `$` or `>` prompts.
  - **Visual Depth:** Use thin, clean borders (1px) and extremely soft, large-radius shadows to create a "floating" effect on the grid.
  - **Canvas Layout:** A structured yet open layout with a persistent Top Navigation and an optional Side Navigation, centering around a main interactive workspace.
- **Vibe:** Professional, organized, high-performance, and "Geek-Chic". Focus on clarity, functional beauty, and a smooth user experience.

## 中文版本

**Prompt (提示词):**

创建一个现代、极客且简约的 Web 界面，灵感源自终端美学和基于节点的协作流。

- **背景：** 使用纯净的白色或极浅灰色背景，辅以细微的点状或线条网格图案（令人联想到蓝图或代码编辑器的画布）。
- **字体：**
  - **UI 与标签：** 使用高质量的等宽字体（如 JetBrains Mono, Fira Code），营造“开发者”专属感。
  - **标题：** 使用粗体无衬线字体（如 Inter, Geist），确保高可读性并形成专业的视觉对比。
- **配色方案：**
  - **主背景：** 纯白色 (#FFFFFF)。
  - **文本：** 深石板灰 (#1E293B) 作为主要内容颜色。
  - **强调色：** 温暖的橙色/桃色 (#F97316 或 #FB923C)，用于交互状态、高亮显示和主要按钮。
  - **辅助元素：** 浅石板色 (#F1F5F9) 用于次要组件的背景。
- **核心组件：**
  - **窗口化节点：** 交互式卡片或节点需设计成简化版系统窗口，顶部左侧带有经典的“红黄绿”三色功能点。
  - **终端式输入：** 搜索框和输入字段应使用命令行隐喻，例如以 `$` 或 `>` 符号开头。
  - **视觉深度：** 使用极细且干净的边框 (1px) 和非常柔和、大半径的阴影，在网格背景上营造出“悬浮”感。
  - **画布布局：** 结构化且开放的布局，拥有常驻的顶部导航栏和可选的侧边栏，核心围绕一个主交互工作区展开。
- **氛围：** 专业、井然有序、高性能且具有“极客范”。注重清晰度、功能美学和流畅的用户体验。
